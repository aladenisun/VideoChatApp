# Video-Web-App

## Project setup

```
npm install
```

### To run Server Side

```
- open two terminal tabs
- 1st Tab: in project folder
    node server.js
```

### To run Frontend Side

```

- 2nd Tab: cd client
    npm run start
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
